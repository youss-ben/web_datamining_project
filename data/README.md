# Data Directory

This directory contains generated data files from the pipeline.

| File | Generated by | Description |
|------|-------------|-------------|
| `crawler_output.jsonl` | `src/crawl/crawler.py` | Raw crawled text from Artemis URLs |
| `extracted_knowledge.csv` | `src/ie/ner_extraction.py` | NER-extracted entity triples |
| `mapping_table.csv` | `src/kg/kb_construction.py` | Entity linking table with confidence |
| `family.owl` | `src/reason/create_ontology.py` | Family ontology for SWRL exercise |
| `family_inferred.owl` | `src/reason/swrl_reasoning.py` | Family ontology after SWRL inference |
| `statistics_report.txt` | `src/kg/kb_expansion.py` | KB size statistics |
| `train.txt / valid.txt / test.txt` | Notebook | KGE datasets (80/10/10 split) |
| `samples/` | — | Sample data for testing |

> **Note**: These files are generated. Run the pipeline scripts to reproduce them.
